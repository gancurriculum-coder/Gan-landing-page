## About the Project

Gan HaChochma is a print-first, academically rigorous Jewish homeschool curriculum designed for families across the full spectrum of traditional observance — Modern Orthodox, Conservative, Mizrachi, Sephardic, and affiliated independents.

It brings together serious Judaic studies and strong secular academics, with a clean secular layer built for families using public charter schools.

**Curriculum components include:**

- Torah & Tanakh (parsha, text study, classical mefarshim)
- Jewish History & Israel Studies (spiral-designed, age-appropriate)
- Hebrew Language (modern Ivrit and tefillah Hebrew)
- Holidays & Halacha (the Jewish calendar as curriculum spine)
- Secular Academics with Jewish integration points
- Multi-age design for families with children across grade levels
